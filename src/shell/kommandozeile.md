---

### Hilfe

```bash
man $BEFEHL

man man

$BEFEHL --help
```

### Verzeichnisse listen / wechseln

```bash
pwd

ls
ls -l  ## listing
ls -lh  ## listing --human-readable
ls -a  ## --all

cd $VERZEICHNIS
cd -  ## zurück ins vorherige Verzeichnis
cd  ## In das $HOME Verzeichnis
```

### Editoren

```bash
nano $DATEI

vim $DATEI
# :q!

vimtutor
```

### Sudo

```bash
sudo $BEFEHL
```
